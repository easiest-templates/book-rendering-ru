# КнижныйМир - Книжный магазин с 3D визуализацией

![Preview](https://cdn.poehali.dev/templates/book-rendering-ru/preview-ru.jpg)

Современный шаблон интернет-магазина книг с интерактивной 3D визуализацией книг на базе Three.js.

## Технологии

- **React 18** + TypeScript
- **Three.js** + React Three Fiber для 3D рендеринга книг
- **Tailwind CSS** для стилизации
- **Radix UI** для компонентов интерфейса
- **Vite** для сборки

## Возможности

- Интерактивная 3D модель книги с вращением
- Адаптивный дизайн
- Каталог книг с фильтрацией
- Карточки товаров
- Корзина покупок

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
